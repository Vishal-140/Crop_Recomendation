## рџљЂ Quick Start (run locally)
### Prerequisites
- **Node.js** (>=14)
- **Python** 3.8+ with `pip`
- (Optional) `virtualenv` for the Python bridge

### Backend
```bash
cd cap_2026/cap_backend
npm install               # install Node dependencies
# Optional: set up a Python virtual environment
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt   # Flask, scikitвЂ‘learn, etc.
npm start                 # starts server on http://localhost:5000
```
### Frontend
```bash
cd cap_2026/cap_frontend
npm install               # install React/Vite dependencies
npm run dev               # starts dev server on http://localhost:5173
```
Open the frontend URL, fill the seven input fields, and youвЂ™ll receive a crop recommendation (emoji + name) based on the trained model.

> **Note:** If the backend cannot locate the `.pkl` files it will exit with an error. Ensure they are present in `model_files` in `.env`.
